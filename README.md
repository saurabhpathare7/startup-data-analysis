🚀 Indian Startup Funding Analysis
An interactive Streamlit Web Application built using Python, Pandas, and Plotly to analyze Indian startup funding data.
This project helps users explore startup investments, investors, funding trends, sectors, cities, and funding rounds through beautiful visual dashboards.
________________________________________
📌 Features
📊 Overall Analysis
•	Total Funding Overview
•	Maximum Funding Raised
•	Average Startup Funding
•	Total Number of Startups
•	Month-on-Month Funding Trend
•	Top Funded Sectors
•	City-wise Funding Analysis
•	Top Startups & Investors
•	Funding Heatmap
🏢 Startup Analysis
•	Startup Industry Details
•	Funding Rounds Information
•	Total Funding Raised
•	Similar Startups Discovery
💰 Investor Analysis
•	Recent Investments
•	Biggest Investments
•	Preferred Investment Sectors
•	Investor Portfolio Insights
